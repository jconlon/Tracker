Creating a new Fair Document

To create a new Fair document. Go to the File menu and launch a New Fair Document Wizard by selecting the menu items:File|New|Fair Document.

Follow the instructions to choose a project, folder and a name for the document. Select Fair as the model object and press Finish to create the document.
