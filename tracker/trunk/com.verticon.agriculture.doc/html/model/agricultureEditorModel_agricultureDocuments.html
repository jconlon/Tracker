Agriculture Documents

The Agriculture Editor enables users to create, transform and edit documents referencing geographic Locations of Livestock Premises and sub locations within a Premises.

Note

Because Agriculture Documents elements references elements in one or more Premises and KML geographic documents, these two document types need to be created before the user creates an Agriculture document.

Agriculture document files utilize a .agri file extension to distinguish them from other types of files.
