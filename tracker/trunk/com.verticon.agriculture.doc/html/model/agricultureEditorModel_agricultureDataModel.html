Agriculture Data Model

Because all user activity centers around the creation and manipulation of Agriculture Documents, understanding the structure (called the data model) of Agriculture documents is the easiest way to learn to use the information management facilities of the Agriculture Editor.

Data Model Core Elements

All Agriculture documents conform to a very simple structure called the Agriculture Data Model. In the Agriculture Data Model, a single Agriculture association is assigned a name and a geographic region. Within this agriculture association there are one or more Location elements. A Location element is a place for assigning livestock premises to geographic locations. The geographic locations assigned to the premises within the Locations may or may not be geographically contained within the Agriculture elements region attribute.
