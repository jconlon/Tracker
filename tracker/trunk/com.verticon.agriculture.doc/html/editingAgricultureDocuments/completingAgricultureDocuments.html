Completing an Agriculuture Document

When KML documents are loaded as resources into Agriculture documents, the editor choice boxes associated with the geography attributes of the Agriculture element and the Location elements will be populated by all Container Features elements from all the loaded KML resources.

Figure 8. Agriculture Project with Loaded Resources


Likewise, when premises documents are loaded as resources into Agriculture documents, the editor choice boxes associated with the livestock attribute of the Location element will be populated by all Premises elements from all the loaded Premises resources.

Once resources are loaded; a name and region can be assigned to the Agriculture association and one or more Locations created.
