Editing Agriculture Documents

Creating a new Document

To create a new agriculture document. Go to the File menu and launch a New Wizard by selecting the menu items:File|New|Other. Figure 4, “New Wizard for Creating a New Agriculture Document”.

Figure 4. New Wizard for Creating a New Agriculture Document


Follow the instructions to choose a project, folder and a name for the document. Select Agriculture as the model object and press Finish to create the document.
