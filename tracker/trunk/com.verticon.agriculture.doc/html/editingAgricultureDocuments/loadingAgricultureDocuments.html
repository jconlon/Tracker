Loading Additional Resources into the Document

Note

Since the Agriculture document references Premises document elements and KML document elements, one or more of these document resources must already exist in the workspace.

To build Location elements in the Agriculture document, the document must first be loaded with the KML and Premsies document resourcess that will be used for the attribute references. To load a resource place the mouse cursor on the root element in the document, right click, and from the pull down menu choose Load Resorce...

Figure 6. Load Resource Dialog


From the dialog choose the KML or the Premises document that you wish to load. Repeat the loading process to load all the resources that will be referenced by the Agriculture document.
