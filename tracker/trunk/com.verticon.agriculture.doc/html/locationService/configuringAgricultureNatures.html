Configuring a Project with an Agriculture Nature

To configure a project with an Agriculture Nature place the mouse cursor on the Project in the Project Explorer right click, and from the pull down menu choose Add/Remove Agriculture Nature.

Figure 10. Load Resource Dialog


Note

A project that has been configured as an Agriculture Nature will be designated with a small capital A on the upper right of the project folder icon.
