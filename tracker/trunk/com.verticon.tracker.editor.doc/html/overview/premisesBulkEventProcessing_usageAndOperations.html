Usage and operations

Adding Events to Manually Selecting Animals

To add a series of events to an animal or set of animals in a Premises Document select the animal or animals and invoke a popup menu item;

  1. Right click on the Animal to get the top level popup menu;

  2. Select the 'Tracker' menu item to get the Tracker popup menu;

  3. Select 'Add Template Events to Selected Animals' to bring up a selection window to select a Template;

  4. Choose an Animal Document Template containing the set of events you wish to add the animal.
