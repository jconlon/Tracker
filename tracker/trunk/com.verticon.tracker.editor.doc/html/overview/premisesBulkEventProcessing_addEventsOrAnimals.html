Add Events and/or Animals to a List of Animals Specified in a File

To add a series of events and/or animals to the Premises as a whole, you will still need to specify which animals to add event to. This can be done indirectly with a Tag IdNumber file. Select a Tag IdNumber file and invoke a popup menu item it.

  1. Right click on the Animal Document Template that contains the list of animal ids which specify the set of animals in the Premises Document that you wish to add events to.

  2. From the popup menu Select 'Tracker' to get the Tracker specific menu.

  3. Select 'Add Template Events to Premises' to bring up a selection window that will allow you to select a Animal Document Template.

  4. Choose an Animal Document Template containing the set of template events and the default animal type you wish to add the file specified animals in the premises.
