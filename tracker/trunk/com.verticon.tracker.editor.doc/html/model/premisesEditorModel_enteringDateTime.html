Entering Date and DateTime

When manually entering Dates and DateTime attribute values in Premises documents the following formating patterns are determinative:

Table 2. Format Patterns for Text Entry

Format PatternExample
yyyy-MM-dd'T'HH:mm:ss'.'SSSZ2007-11-12T00:00:00.000-0600
yyy-MM-dd'T'HH:mm:ss'.'SSS2007-11-12T00:00:00.000
yyyy-MM-dd'T'HH:mm:ss2007-11-12T00:00:00
yyyy-MM-dd'T'HH:mm2007-11-12T00:00
