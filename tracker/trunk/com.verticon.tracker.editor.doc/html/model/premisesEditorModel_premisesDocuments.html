Tracker Premises Documents

Whether it is word processing documents, spreadsheet documents, or presentation documents, today's computer users are most familiar with software applications that create and manipulate documents that describe specific instances of information.

The Tracker enables users to create, transform and edit Livestock Premises documents that represent specific Animal Premises. Premises documents are displayed and manipulated with Tracker Editors, Wizards, and utilities. The attributes of the premises, the animals in the premises and the events associated with these animals are specified with Premises documents.

Tracker Model document files utilize a .premises file extension to distinguish them from other types of files.
