Overview Tracker Premises Editor

What is the Tracker Premises Editor?

Tracker Premises Editor is an Open Source, Eclipse Modeling Facility EMF based editor Software Application Framework for the development of Multi-platform Livestock Information Solutions.

Project Goals

The Tracker Premises Editor project was undertaken to provide;

  1. A useful documentation tool for livestock producers and infrastructure providers;

  2. A low cost documentation solution for deployment to both large and small scale livestock operations;

  3. An industry standard format for livestock event history inter-exchange.

Functional Description

Tracker Premises Editor documents animal event histories for farm animal producers and/or livestock infrastructure providers. Animal premises, animals and event histories are created, edited, manipulated, and persisted as Premises documents. Premises documents are displayed and manipulated with Tracker editors, wizards, and various importing and exporting utilities.

Animals and Events are added to Tracker Premises Documents;

  1. Individually by the user via the Tracker Premise Document Editor and Properties Viewer;

  2. In bulk, by the user, through Tracker's Bulk Event Processing Utilities;

  3. In real time, from RFID readers, through the Tracker Transaction Processing Facility.
