Tracker Lite - Smartphone Application

Tracker Lite is an easy to use smart phone application that provides a convenient way for end users to record and archive animal life data and event histories for a livestock premises.

Tracker Lite Features

Tracker Lite

  • Import and upload Animal Life and Animal histories from the Tracker Cloud service;

  • Acquire event data from associated RFID readers;

  • Utilize standard event or site specific generic events;

  • Acquire Geolocation data from associated GPS receivers;

  • Work in concert with other Tracker Lite or Tracker Professional users;

  • Inter-exchange premises document information between phones.
