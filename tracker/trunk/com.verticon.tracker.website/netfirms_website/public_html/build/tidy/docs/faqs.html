3. Getting Started

3.1. Q: I have installed the software what do I do next?
3.2. Q: When I first start the Tracker I get a Welcome Screen. What should I do?
3.3. Q: Why do I have to provision the software after install?
3.1.

Q: I have installed the software what do I do next?

A: It is always best to read the documentation first. Did you already do this? If not start with the Eclipse Workbench User Guide and then read the Tracker User Guide.

Note that both of these documents are available from within the Tracker. Go to the Help > Help Contents to show the built in Help Documentation.

3.2.

Q: When I first start the Tracker I get a Welcome Screen. What should I do?

A: Click into the various icons to view the contents of the Welcome information. Then go to first steps page and provision the tool.

3.3.

Q: Why do I have to provision the software after install?

A: Tracker BI Toolkit is a component container that is customizable by the end user. You can select and provision the Tracker Toolkit only with the features to meet your needs.

4. iRouter

4.1. Q: What is the relationship between the Tracker BI Toolkit and the iRouter?
4.1.

Q: What is the relationship between the Tracker BI Toolkit and the iRouter?

A: Tracker BI Toolkit is a container for tool features. When it is first installed it is empty. For the Tracker BI Toolkit to be useful the user must select and install the tool features they require. The iRouter is one of the many Tracker tool features that can be installed. The end user can select and install iRouter features by itself or with a combination of any of the other Tracker livestock information management tool features.

5. Custom Use Cases

5.1. Q: I am not in the US can I still use Tracker?
5.2. Q: Is Tracker Internationalized?
5.1.

Q: I am not in the US can I still use Tracker?

A: Yes, although the Tracker Professional Services and Component Development Teams will have to craft a data model and editor feature for your use case.

5.2.

Q: Is Tracker Internationalized?

A: Tracker is not internationalized in milestone3 but will be as part of the beta program.
