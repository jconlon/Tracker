Introduction
TRENDexplorer is one of a family of network management integration applications available from Verticon, Inc.

TRENDexplorer is a support  tool for analysts and consultants who work with the Trinagy TRENDTM network performance reporting application.  TRENDexplorer provides the analyst or consultant with an easy way to explore TREND database repositories directories and data tables. 

Graphically navigating the TREND System device and report pack directories with TRENDexplorer makes troubleshooting TREND reporting issues and looking up specific performance data significantly easier than using scripts or directly entering SQL queries. 


Description
 TRENDexplorer is comprised of a set of six repository navigators for extracting specific device performance data residing in one of the many TREND performance data tables and a spreadsheet-like table for exploring the data once it has been extracted.

Device Navigation Controls:

  • Text Field - type ahead sensitive text field for quick keyboard navigation of TREND devices;
  • Scrollable list control -  for fast lookups of devices sorted alphabetically;
  • Scroll enabled tree control -  for navigating device contained in TREND device group hierarchies

Table Navigation Controls:

  • Text Field - type ahead sensitive text field for quick keyboard navigation of TREND data tables
  • Scrollable list control -  for fast lookups of data tables sorted alphabetically;
  • Scroll enabled tree control -  for navigating tables contained in TREND report pack hierarchies

Data WorkSheet:

  • shows contents of all data table columns;
  • column based sorting;
