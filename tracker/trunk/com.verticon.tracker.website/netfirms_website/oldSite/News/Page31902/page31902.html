VERTICON, INC. ANNOUNCES BETA TEST OF ITS INTERACT2 PRODUCT, SEEKS BETA TEST SITES. 

ATLANTA, GA, September 11, 2000 – Verticon, Inc. announces beta test availability of its InterAct2 product.  The InterAct2 Desktop Integration Suite is a network management integration product that allows users to work with network management data from many sources. The product provides the ability to export, visualize, import, transform and update data from relational databases, XML and delimited text files.

Verticon President John Conlon stated, "Integrators, analysts, consultants and operations staff often must integrate data from several network management applications. These applications contain a wealth of information, which, if more easily available, could enhance these applications' value and create opportunities to better manage the underlying networks. Verticon is actively seeking beta testers to assist us in bringing the product to general availability."

The InterAct2 Desktop Integration Suite provides an application to launch integration documents. Several document sets are provided with the product, and the user may modify them or build his own documents in order to expand product functionality.

Out of the box, Verticon provides integration document Catalogs for three main purposes:

  1. View and manipulate data for TREND®, a product from Desktalk Systems, Inc.;
  2. View discovered nodes and events from OpenView®/NNM, a product from HP®; and
  3. Import nodes and events from NNM into TREND.
