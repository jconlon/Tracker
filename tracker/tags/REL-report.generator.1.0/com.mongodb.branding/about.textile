h1=. MongoDB Drivers

com.mongodb Version 2.7.2

h2. Introduction

"MongoDB":http://www.mongodb.org/ (from "humongous") is a scalable, high-performance, open source NoSQL database. The MongoDB Java Drivers provide programmatic access to the MongoDB data store.
